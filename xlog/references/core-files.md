# xlog 核心文件

## LOG-RULES.md（规范类）

- 简介: 日志规范，记录 Logger 列表、级别用法、代码模式、禁忌
- 搜索策略: 文件名含 log, 日志规范, logging
- 格式规范: `references/log-rules-format.md`
- 状态字段: `log_rules`

## LOG-COVERAGE.md（追踪类）

- 简介: 日志覆盖度，跟踪每个模块的日志扫描状态和盲区
- 搜索策略: 文件名含 coverage, 覆盖
- 格式规范: `references/log-coverage-format.md`
- 状态字段: `log_coverage`

# LOG-COVERAGE.md 规范

## 文件格式

```markdown
# LOG COVERAGE

> 由 /xlog skill 自动维护

更新时间：YYYY-MM-DD

## 概览

| 状态 | 数量 |
|------|------|
| ✅ 达标 | 0 |
| ⚠️ 有问题 | 0 |
| ⏳ 未扫描 | 0 |

## 模块明细

| 模块 | 状态 | 扫描日期 | 盲区数 | 不规范数 | 备注 |
|------|------|---------|--------|---------|------|
```

## 模块划分

从项目目录结构和代码组织动态推导模块（如按目录、按功能区域），不硬编码。

## 状态判定

- **✅ 达标**：覆盖充分且全部符合规范
- **⚠️ 有问题**：有盲区或不规范的日志（盲区数 + 不规范数 > 0）
- **⏳ 未扫描**：尚未被 `/xlog` 扫描过
